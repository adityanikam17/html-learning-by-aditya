Tables :

Tables in HTML are used to represent real life able data.

  • tr : is used to display table row.
  • td : is used to display table data.
  • th : is used to display table header.
  • border : is used to mark border in table.
  • thead : used to wrap table header.
  • tbody : used to wrap table body.
  • tfoot : used to wrap footer
